# Dégustation Bret's

Petite appli de jeu pour une soirée dégustation à l'aveugle de 59 chips Bret's, répartie en 5 manches. Les joueurs devinent la saveur de chaque chips et notent leur préférence ; un rôle "maître du jeu" permet de corriger les réponses et de suivre le classement en direct.

## Fichiers

- `DegustationBrets.jsx` — composant React principal (appli joueur + appli maître du jeu).
- `fiche-degustation-brets.html` — fiche imprimable (pass dégustation) au format HTML autonome.
- `degustation-brets-offline.html` — version HTML/CSS autonome jouable hors-ligne, sans dépendances.
